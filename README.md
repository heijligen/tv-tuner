# tv-tuner

This is an auxiliary application to simplify the portation of [hdtv](github.com/heijligne/hdtv) to MacOS and Windows

## How-To
```
git clone https://github.com/heijligen/tv-tuner
cd tv-tuner
uv venv --system-site-packages
uv run tv-tuner
```
The output should end with a line saying `ok`
